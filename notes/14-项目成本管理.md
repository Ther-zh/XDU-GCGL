## 本章缩写对照（EVM 挣值管理）

> **EVM** = Earned Value Management。本章 **PV** = Planned Value（计划值），非经济章 Present Value。

| 缩写 | English | 中文 |
|------|---------|------|
| **PV** | Planned Value | 计划值 |
| **EV** | Earned Value | 挣值 |
| **AC** | Actual Cost | 实际成本 |
| **BAC** | Budget at Completion | 完工预算 |
| **CV / SV** | Cost / Schedule Variance | 成本 / 进度偏差 |
| **CPI / SPI** | Cost / Schedule Performance Index | 成本 / 进度绩效指数 |
| **EAC / ETC** | Estimate at Completion / To Complete | 完工估算 / 尚需估算 |
| **WBS** | Work Breakdown Structure | 工作分解结构 |

## 知识点精讲

### 14.1 成本管理过程

#### 【★★☆】估算 vs 预算 vs 基准

| 概念 | 含义 |
|------|------|
| **成本估算** | 对完成活动所需资源的定量评估 |
| **成本预算** | 把估算汇总并分配至各 WBS，形成**成本基准** |
| **成本基准** | 经批准的按时间分段的预算，不含管理储备 |

#### 【★★☆】与工程经济的区别

| | 工程经济分析 | 项目成本管理 |
|---|--------------|--------------|
| 阶段 | 立项前/方案比选 | 执行与监控 |
| 焦点 | 投资可行性、NPV/IRR | 预算执行、偏差控制 |
| 工具 | 现金流量、评价指标 | 挣值分析、S 曲线 |

#### 【★★☆】与进度管理的关系

共用 WBS 与进度基准；挣值法**同时**反映进度与成本绩效（SPI、SV）。

### 14.2 成本曲线

#### 【★★☆】成本负荷图

显示各时段计划支出，用于资源平衡。

#### 【★★☆】S 曲线（成本累计曲线）

横轴时间、纵轴累计成本；计划线 vs 实际线对比，直观显示超支/节约趋势。

### 14.3 挣值分析（必算）

#### 【★★★】三个基本量

| 符号 | 名称 | 含义 |
|------|------|------|
| **PV** | 计划值 | 按进度计划**应完成**工作的预算成本 |
| **EV** | 挣值 | **实际完成**工作的预算成本 |
| **AC** | 实际成本 | 完成工作实际发生的成本 |

#### 【★★★】四个绩效指标

```
CV = EV − AC     （成本偏差，<0 超支）
SV = EV − PV     （进度偏差，<0 落后）
CPI = EV / AC    （成本绩效，<1 超支）
SPI = EV / PV    （进度绩效，<1 落后）
```

#### 【★★★】完工估算

典型公式：**EAC = BAC / CPI**（假设当前成本绩效延续至完工）

**ETC** = EAC − AC

#### 【★★★】做题流水线

抄 PV/EV/AC/BAC → 算 CV、SV → 算 CPI、SPI → 写成本/进度结论句 → 若求 EAC 用 BAC/CPI。

## 例题：课后习题 Q8（必练）

**已知**（项目进行 2 个月）：PV = 23 万，EV = 20 万，AC = 25 万，BAC = 120 万。

| 指标 | 计算 | 结果 | 解读 |
|------|------|------|------|
| CV | 20−25 | **−5 万** | 超支 5 万 |
| SV | 20−23 | **−3 万** | 进度落后（少完成 3 万预算工作量） |
| CPI | 20/25 | **0.80** | 每花 1 元只完成 0.8 元预算工作 |
| SPI | 20/23 | **≈0.87** | 进度约为计划的 87% |

**(2) 结论**：**落后于进度计划**，**超出预算**。

**(3) EAC** = 120 / 0.80 = **150 万** → 比计划执行**差**。

**(4) 进度**：若按 SPI 外推，总工期约为原计划 / SPI ≈ 延长；完成得**晚于**预期。

## 本章小结

1. 成本基准是挣值对比的「尺子」。
2. CPI、SPI 同时看「钱花得值不值」「活干得够不够快」。
3. 索契奥运会等案例：成本超支与监控不力。
4. 期末计算题高频：给 PV/EV/AC/BAC 求四指标 + 判读 + EAC。
